A simple single-page AJAX-based Twitter clone that uses jQuery,HTML5 and CSS3

## Dependencies

- Express
- chance
- nodemon
- body-parser 
- md5


## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm run local` command.
